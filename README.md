### [GankIO](http://gank.io/) Android 客户端 感谢daimajia 提供Api支持！

### App
MVP+Dagger2.android+Retrofit+RxJava2+Glide+Realm

### ScreenShot


### 功能
1. GitHub 账户即可登录，无需密码，用户创建本地账户收藏干货用。
2. 侧滑收藏／取消收藏。
3. 每日干货浏览，分类浏览。
4. 搜索功能，搜索历史支持。
5. 高清妹子长按下载
6. 干货分享（原生）
